Run multilingual e-commerce sites with Easy Digital Downloads and WPML

== Description ==

EDD Multilingual lets you run light-weight and digital shops, with electronice-only delivery, in multiple languages.

This is a glue plugin, that uses hooks in both [Easy Digital Downloads](http://wordpress.org/extend/plugins/easy-digital-downloads/) and [WPML](http://wpml.org).

It will let you run a complete shop and easily translate everything.

== Installation ==

1. Upload 'edd-multilingual' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress


== Frequently Asked Questions ==

= Does this plugin do anything by itself? =

This is a glue plugin between EDD and WPML. You need to have both in order for this plugin to do anything.
